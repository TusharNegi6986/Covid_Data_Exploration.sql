# COVID-19 Data Exploration using SQL

## Project Overview

This project focuses on exploring COVID-19 data using SQL to uncover insights about infection rates, death rates, and vaccination progress across different countries and continents.

The goal of this project was to strengthen my SQL skills through real-world data analysis and understand how data exploration works in practical scenarios.

This project is based on the SQL portfolio project by [Alex The Analyst](https://www.youtube.com/@AlexTheAnalyst?utm_source=chatgpt.com).

---

## Dataset Used

The dataset contains:

* COVID deaths data
* COVID vaccinations data

Key columns include:

* Location
* Date
* Population
* Total Cases
* New Cases
* Total Deaths
* New Deaths
* New Vaccinations

## Questions Explored

### 1. Total Cases vs Total Deaths

* What percentage of infected people died?

### 2. Total Cases vs Population

* What percentage of the population got infected?

### 3. Highest Infection Rate by Country

* Which countries had the highest infection percentage?

### 4. Highest Death Count by Country

* Which countries had the highest death counts?

### 5. Highest Death Count by Continent

* Which continents were affected the most?

### 6. Global Daily Trends

* How did cases and deaths change over time?

### 7. Rolling Vaccination Count

* How many people were vaccinated over time?

### 8. Percentage of Population Vaccinated

* What percentage of people got vaccinated?
